## 🇧🇷 Português

Instalação profissional do Portainer utilizando Docker Compose.

### Características:

- Volume nomeado para persistência
- Network dedicada
- Restart policy configurada
- Estrutura pronta para integração com Nginx Proxy Manager

### Subir o ambiente:

```bash
docker compose up -d
```

Acesse:

```
https://IP_DA_VPS:9443
```
